🇮🇳 India BPT Colleges Directory

A comprehensive, state-wise directory of Bachelor of Physiotherapy (BPT) colleges across India. This repository aims to help aspiring physiotherapy students find accurate and organized information about BPT institutions, admissions, fees, affiliations, and official resources.

📌 About the Project

Choosing the right BPT college can be challenging due to scattered information across multiple websites. This repository brings everything together in one place by providing a structured list of BPT colleges from all Indian states and union territories.

Whether you're a student, parent, educator, or healthcare professional, this repository can serve as a valuable reference.

🎯 Objectives

Provide a state-wise list of BPT colleges in India.
Share essential college information in a structured format.
Help students compare institutions easily.
Create a free and open-source resource for the physiotherapy community.

📂 Information Included
Each college entry may contain:
🏫 College Name
📍 Location
🏛️ Affiliated University
🎓 Degree Offered
💰 Approximate Fees
📅 Admission Process
📞 Contact Information
🌐 Official Website
